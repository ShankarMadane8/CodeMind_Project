# CodeMind Project

## Project Overview

The CodeMind project is a web application with the following components:

- **Backend**: 
  - Technologies: Spring Boot, MySQL
  - Entities: Customer, Engineer, Manager, CustomerFeedback
  - CRUD operations implemented for Customer, Engineer, Manager, and CustomerFeedback entities.

- **Frontend**:
  - Technology: Angular
  - Features: Pages for login for three user types (Customer, Engineer, Manager) and a dashboard.
